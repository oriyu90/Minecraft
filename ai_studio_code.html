セキュリティの注意事項

このWebアプリは完全にブラウザ上で動作します。入力したパスワードは外部サーバーには送信されず、お使いのブラウザのローカルストレージ（localStorage）のみに保存されます。

Web APIは暗号化されていない HTTP 通信のため、外出先からアクセスする際はポート開放ではなく、Tailscale などの VPN 接続を経由することを強く推奨します。
