セキュリティとHTTPS制限について

GitHub Pages などの HTTPS サイトから、ローカルの http:// 接続を呼び出すと、ブラウザのセキュリティ制限 (Mixed Content ブロック) により通信が失敗します。

この問題を回避するには、以下いずれかの方法を推奨します。

  • Cloudflare Tunnel や Tailscale Funnel を使用してWeb APIを HTTPS 化し、上記の「接続先ベースURL」にその https:// URLを入力する。
  • Safariブラウザを使用する（Safariは条件によってローカルIPへのHTTP通信をブロックしない場合があります）。
  • アプリが内蔵しているローカルのWebサーバー（直接 http://<MacのIP>:<ポート>）をブラウザで開いて管理する。
